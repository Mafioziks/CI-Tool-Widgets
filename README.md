# CI-Tool-Widgets

## About

Tools are created to easyer and faster use CI Tools
Project can work with:
* Jira
* Jenkins
* Gitlab

## Requirements
python 3
packages:
* termcolor
* jira
* jenkinsapi
* pygobject

Need to be installed
* PyGTK = 3.0
* AppIndicatos3 = 0.1
* Notify = 0.7

## Usage
Run JiraHelper.py

### Add product links
Indicator menu -> Settings

### Manage presets for Jenkins
Indicator menu -> Presets
